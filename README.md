# 🧠 Flashcard Study App
The Flashcard Study App is a mobile application designed to help users study and learn new information using flashcards. The app allows users to create, edit, and study flashcards, and provides features such as filtering, searching, and tracking progress. The app is built using Flutter and utilizes a variety of packages, including Go Router and Google Fonts.

## 🚀 Features
* Create, edit, and study flashcards
* Filter and search flashcards
* Track progress and view statistics
* Customize the app's theme and appearance
* Use Go Router for client-side routing
* Utilize Google Fonts for typography

## 🛠️ Tech Stack
* Flutter
* Go Router
* Google Fonts
* Sizer
* Flutter SVG
* Shared Preferences
* Flutter Lints

## 📦 Installation
To install the Flashcard Study App, follow these steps:
1. Clone the repository using `git clone`
2. Run `flutter pub get` to install dependencies
3. Run `flutter run` to launch the app

## 💻 Usage
To use the Flashcard Study App, follow these steps:
1. Launch the app and create a new flashcard deck
2. Add flashcards to the deck and customize their appearance
3. Study the flashcards and track your progress
4. Use the app's filtering and searching features to find specific flashcards
